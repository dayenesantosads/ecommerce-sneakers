# 📌 Projeto E-commerce Sneakers

## 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de um site de e-commerce com foco em sneakers, utilizando apenas HTML e CSS. O objetivo principal é simular uma loja virtual simples, com navegação entre páginas e apresentação de produtos.

O projeto foi desenvolvido como atividade prática da disciplina de Programação Web, com foco na estruturação de páginas e estilização visual.

## 🎯 Objetivo

Criar um site funcional utilizando apenas HTML e CSS
Aplicar conceitos de estruturação semântica
Trabalhar com estilização e layout responsivo básico
Simular um ambiente de e-commerce
🛠️ Tecnologias Utilizadas
HTML5
CSS3

## 🧩 Estrutura do Projeto

O site é composto por 4 páginas principais:

### Página Inicial (index.html)

Contém:

- Dois produtos em destaque
- Uma notícia relacionada ao universo sneakers

### Produtos (produtos.html)

- Lista de produtos disponíveis para compra

### Carrinho (carrinho.html)

- Simulação de uma página de carrinho

### Contato (contato.html)

- Formulário para contato com o usuário

## 🎨 Funcionalidades

- Navegação entre páginas
- Exibição de produtos
- Layout organizado com CSS
- Formulário de contato (não funcional, apenas visual)

## ▶️ Como Executar o Projeto

Faça o download ou clone este repositório:

git clone https://github.com/dayenesantosads/ecommerce-snearkers.git

Abra a pasta do projeto
Execute o arquivo index.html em qualquer navegador
(ou utilize a extensão Live Server no VS Code)

### 📌 Observações

Este projeto não possui integração com banco de dados
Não há funcionalidades de compra real
O foco é exclusivamente front-end
